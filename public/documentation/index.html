Introduction

Rise - Bootstrap UI Kit

Rise is a modern Bootstrap 4 UI Kit, designed and developed with attention to detail. Includes 200+ reusable components, 50+ pre-built sections and 6 pre-built templates.

Rise UI Kit is not just easy to use, but is also very easy to customize, building a web-app a website, changing colors and styling never been easier. We redesigned most of the default Bootstrap components, added many custom classes, utility classes, custom elements and plugins to make a beautifully designed, very functional and feature-rich UI Kit.

Setup SASS

To use SASS you will have to use a SASS Processor application, like Scout App, it is available for Linux 32 / 64bit, Mac and Windows, it requires project setup in order to process the SASS files from a set folder or you can also use Prepros, it is also available for Linux, Mac and Windows and it is a bit easier to setup application.

Change Variables

To edit any theme file you will need a Text or Code Editor, we recommend using Sublime Text, as it is the Text Editor of our choice to write code, but every Text or Code Editor will work fine to edit any theme files.

To change the theme's fonts, colors and other variables, please open the _variables.scss file from assets / scss folder and modify the font-family to your preferred font, font-weights, the letter-spacing values, change color codes and you can also add custom variables of any type if needed.

Utility Classes

We've added a lot of utility classes ( CSS Classes ) to our theme to make the theme creation and customization process easier, using the built-in utility classes you can easily change or add margin, padding, border, color, animation, etc. to any element by just adding a CSS class in HTML. You can find and modify all utility classes in assets / scss / _utility.scss file.

Spacer Classes

Adding .mt-0, .mt-5, .mt-10, ... .mt-50 in size jumps of 5 ( 5px ), then .mt-60, .mt-70, ... .mt-100 in size jumps of 10, then .mt-150 and .mt-200 represents margin-top-(value), the values are consistent over all different variations of the margins and paddings, for all-around margin use .m-(value) class, margin-bottom is .mb-(value), margin-left is .ml-(value), margin-right is .mr-(value), we also added vertical margins ( margin-top and margin-bottom of the same size ), with .m-y-(value) and horizontal margins with .m-x-(value). The same repeats with paddings, but now instead of .m we will use .p, some examples for every direction:

padding: 0; = .p-0

padding-top: 5px; = .pt-5

padding-bottom: 10px; = .pb-10

padding-left: 15px; = .pl-15

padding-right: 20px; = .pr-20

padding-top: 25px; padding-bottom: 25px; = .p-y-25

padding-left: 30px; padding-right: 30px; = .p-x-30

Colors

Adding color classes to any element will affect the element's background, text and border colors, below can be seen some of the background-color bg-(color-name):

background-color: $primary = .bg-primary

color: $primary = .text-primary

border-color: $primary = .border-primary

We have many more utility classes in assets / scss / _utility.scss file for font-sizes, font-weight, line-height, images, spacers, opacity, borders, positioning, position values and much more, also it is possible to add more custom utility classes with any value by simply specifyng the class and the value in _utility.scss file, it is recommned to add the new classes to the bottom of the file and using !important after every CSS property to surely override other settings.

Global Settings

Set your font-family by including the font's embed code into the <head> of your HTML document. The default font we used is a Google Font

After embedding the font-family code in the HTML Document, the font-family must be specified in CSS, in the theme's assets / scss / _variables.scss file:

$title-font: 'Barlow', sans-serif;
$primary-font: 'Barlow', sans-serif;
$secondary-font: 'Roboto Slab', serif;

Accordions

How it works

The collapse JavaScript plugin is used to show and hide content. Buttons or anchors are used as triggers that are mapped to specific elements you toggle. Collapsing an element will animate the height from its current value to 0. Given how CSS handles animations, you cannot use padding on a .collapse element. Instead, use the class as an independent wrapping element.

ACCORDION
Anim pariatur cliche reprehenderit, enim eiusmod high life accusamus terry richardson ad squid. 3 wolf moon officia aute, non cupidatat skateboard dolor brunch.

        </div><!-- / card-header -->

        <div id="collapse1" class="collapse" role="tabpanel" aria-labelledby="heading1">

Progress

How it Works

Progress components are built with two HTML elements, some CSS to set the width, and a few attributes. We don’t use the HTML5 <progress> element, ensuring you can stack progress bars, animate them, and place text labels over them.

Animations

Using animate.css you can add animations using utility class, like .animated.fadeInLeft followed by another utility class which is created to order the animations, they are from .first to .tenth

Put that all together, and you have the following examples.

Label: 70%
<div class="progress" style="height: 5px">
    <div class="progress-bar bg-primary animated fadeInLeft first" role="progressbar" style="width: 50%; height: 5px;" aria-valuenow="50" aria-valuemin="0" aria-valuemax="100"></div>
</div><!-- / progress -->

<div class="progress">
    <div class="progress-bar bg-success animated fadeInLeft second" role="progressbar" style="width: 60%" aria-valuenow="60" aria-valuemin="0" aria-valuemax="100"></div>
</div><!-- / progress -->

</div><!-- / progress -->
